Detailed Description

Class to load Physically Based Materials from the PhysicallyBased site.

The class can convert the materials to MaterialX format for given target shading models.

Convert the Physically Based Materials to MaterialX format for a given target shading model.

Parameters
materialNamesThe list of material names to convert. If empty, all materials will be converted.
shaderCategoryThe target shading model to convert to. Default is 'standard_surface'.
remapKeysThe remapping keys for the target shading model. If empty, the default remapping keys will be used.
shaderPreFixThe prefix to add to the shader name. Default is an empty string.
Returns
